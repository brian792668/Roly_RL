Roly:     Real robot code file, which is used to test code on real Roly.
Roly_XML: XML file which can be used in Mujoco.

RL_arm: RL training/testing file
	model1: moving model
	model2: prepositioning model
	
natural_poasture:
	models:   Used to store the trained models.
	markers:  Used to label the natural poses.
	datasets: Used to store the labeled data and could be used to train NP models.
